# 📊 XML-Reader
 
O XML-Reader  é uma ferramenta em Python para a leitura, visualização e exportação de dados de arquivos NFe (Nota Fiscal Eletrônica), MDFe (Manifesto Eletrônico de Documentos Fiscais) e CTe (Conhecimento de Transporte Eletrônico).

O objetivo deste projeto é transformar a gestão de arquivos XML complexos em um dashboard organizado, permitindo a exportação rápida para formatos de relatório amplamente utilizados (Excel e PDF).

## ✨ Funcionalidades
🔍 Leitura Inteligente: Suporte nativo para arquivos XML de NFe, MDFe e CTe.

📈 Dashboard em Tempo Real: Visualização dinâmica do valor total acumulado e contagem de documentos importados.

🖥️ Interface Moderna: Construída com CustomTkinter, oferecendo um visual limpo, suporte a DPI alto e uma experiência de usuário fluida.

📊 Exportação para Excel: Gera relatórios .xlsx compatíveis com planilhas para análises detalhadas.

📕 Exportação para PDF: Gera relatórios formatados em modo paisagem (A4), ideais para impressão ou envio por e-mail.

🖱️ Navegação Fluida: Tabela com rolagem horizontal e vertical sincronizada para lidar com grandes volumes de dados.

## 🛠️ Tecnologias Utilizadas
- Python: Linguagem base do projeto.

- CustomTkinter: Interface gráfica moderna (UI/UX).

- Pandas: Processamento de dados e exportação para Excel.

- ReportLab: Geração de documentos PDF profissionais.

- ElementTree (XML): Parsing eficiente da estrutura de arquivos fiscais brasileiros.
